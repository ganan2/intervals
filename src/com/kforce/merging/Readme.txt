To include Junit library into the module:
Eclipse: Right click on the project - build path -  add libraries - JUnit

reset() cannot be placed in setup() or tearDown() or else it results in StackOverflow Exception. 

A list of zipcodes is created and populated in the main method. To get the output, call the method getSortedAndMergedIntervals(List<Interval> globalList) defined in the class of MergeIntervals.
